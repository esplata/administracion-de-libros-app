# Administracion de Libros

[en]
This project was thought to fix, apply, confirm what learned during SNE - Nafar Lasarte Bootcamp "Albañiles Digitales - Sortware builders".
Is a very siple REA application.
There is a menu, and two option: add book, list books.
Each inserted book will be recorded on a CoachDB database.
